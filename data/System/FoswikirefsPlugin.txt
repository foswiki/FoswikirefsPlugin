---+!! Foswikirefs Plugin

%SHORTDESCRIPTION%

%TOC%

This plugin implements a single macro: %REV2REF.  It is passed a Foswiki
revision number in the old Subversion repository.  It maps that revision to a
git hashref that can be viewed on Github.com.



---++ Preferences

This plugin has no preferences or configuration settings.
